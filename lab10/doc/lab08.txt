1.2 Загальне завдання

Для лабораторної роботи “Функції” необхідно додати можливість генерації Doxygen
документації.

Розробити повноцінний звіти для лабораторної роботи “Функції”, що присвячена функціям
у двох форматів (+їх репрезентація у PDF форматі)

2. Хід роботи

Перелік пакетів/утиліт, що були встановлені в контексті цієї роботи:

2.2 Початок роботи.

До файлу Makefile були внесені зміни, для побудови html сторінки зі звітом програми

+ doxygen: 
+ 	doxygen Doxyfile
	
all: clean prep compile doxygen
